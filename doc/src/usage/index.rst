Usage
=====

Installation
------------

End-user installation
*********************

.. code-block:: console

   (.venv)$ pip install abrain

Precompiled wheels are available (see
https://github.com/kgd-al/abrain/wiki/wheels for the full list)

Editable install
****************

.. code-block:: console

   $ git clone ...
   $ ./commands.sh install-dev
   OR
   $ ./commands.sh install-dev-tests
